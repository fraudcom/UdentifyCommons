# UdentifyCommons SDK

**Important:** This repository solely hosts the `Package.swift` manifest file, enabling easy integration of the Commons SDK into your Swift projects using Swift Package Manager (SPM).

The UdentifyCommons SDK is a foundational library that provides essential utilities and common functionalities used across our multiple SDKs.
